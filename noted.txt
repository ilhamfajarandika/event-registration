Main Feature
1. User Authentication
    User registration with email & password
    Login & Logout
    Detail & update profile

2. Event Information (Single Event)
    Show event Detail
    date, location, dan event description

3. Registration System
    User registration for event
    Generate QR Code for e-ticket
    Check-in user (admin)

4. Dashboard
    User dashboard: registration status & e-ticket
    Admin dashboard: list user & check-in

=================================================================================

Endpoint API
1. Authentication
    POST /api/auth/register
    POST /api/auth/login
    POST /api/auth/logout
    GET /api/auth/me
    PUT /api/auth/me

2. Events
    GET     /api/events                  # Get all events

3. Registrations
    POST /api/registrations     # Register to event
    GET /api/registrations/my   # Registration for user
    GET /api/registrations/:id  # Detail registration (e-ticket)

4. Admin
    GET /api/admin/registrations                # User list
    PUT /api/admin/registrations/:id/check-in   # User Check-in
